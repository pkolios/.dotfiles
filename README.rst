.dotfiles
=========

Install dotfiles
----------------
In the home dir

::

    git clone git@github.com:pkolios/.dotfiles.git ./dotfiles
    cd ~/.dotfiles
    make

Uninstall dotfiles
------------------

::

    cd ~/.dotfiles
    make clean
